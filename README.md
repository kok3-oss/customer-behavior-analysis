# Customer Behavior Analysis

## Project Overview

This project analyzes customer transaction data to identify customer segments, purchase behavior patterns, and churn risks. The analysis uses feature engineering and customer segmentation techniques to provide actionable business insights.

## Objectives

* Clean and prepare customer transaction data
* Perform feature engineering
* Segment customers using RFM Analysis or Clustering
* Analyze purchase patterns and retention trends
* Identify potential churn risks
* Provide recommendations to improve customer engagement

## Dataset

Customer Behavior Analysis Dataset from Kaggle.

### Data Cleaning

* Duplicate removal

### Feature Engineering

### Customer Segmentation

* RFM Analysis
* Customer Clustering

### Visualizations

* Customer segment distribution
* Purchase frequency trends
* Revenue contribution by segment
* Retention and churn analysis

## Key Findings

* Identified high-value loyal customers.
* Detected at-risk customers with declining activity.
* Found customer segments with different purchasing behaviors.
* Highlighted opportunities for targeted marketing campaigns.

## Recommendations

1. Reward loyal customers through loyalty programs.
2. Re-engage at-risk customers with personalized offers.
3. Use targeted campaigns for different customer segments.
4. Improve retention through regular engagement.
5. Monitor churn indicators and intervene early.
